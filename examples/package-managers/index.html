Installation Examples

npm install mailto-handler

npm is the default package manager for Node.js and is widely used.

yarn add mailto-handler

Yarn is a fast, reliable, and secure alternative to npm.

pnpm add mailto-handler

pnpm is a fast, disk space efficient package manager.

bun add mailto-handler

Bun is an all-in-one JavaScript runtime and toolkit designed for speed.

Usage Example

The library works the same way regardless of which package manager you use:

// ES Module import
import MailtoHandler from 'mailto-handler';

// Initialize with options
const handler = new MailtoHandler({
  theme: 'auto',
  autoDetect: true,
  rememberChoice: true
});
